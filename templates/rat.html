Usage

The Registration Analysis Tool requires ONE .csv/.xlsx/xls file. OPTIONALLY you can enter the store name you wish to get analysis for (e.g KFC, Boston Market Corporation, Taco Bell). This will default to Dollar General if left empty. The inputted spreadsheet MUST contain columns with the names: Device Type, Name1 & Name2. The column TCP? is also used to generate the TCP data but is not strictly required. (Older months did not have a TCP? column)
